AlgoTrade – Automated Buy Strategy

Project Period: April 2025 – Present

This project is a C#-based algorithmic trading strategy implementation designed to execute buy trades on financial instruments with robust risk management features such as stop loss and take profit orders.

Built as part of a modular trading bot framework, the strategy integrates with MetaTrader 5 and uses real-time market data to manage trades. It ensures that only one trade is open at a time, reducing risk exposure and improving return optimization.

Key Features

  • Focused buy-side trading strategy that opens one trade at a time per instrument
  • Configurable stop loss and take profit levels using pip distance
  • Real-time market quote fetching and price calculations
  • Integration with MetaTrader 5 API for trade execution and management
  • Automated trade lifecycle: entry, exit, and adjustment
  • Trailing stop loss support to secure profits
  • Robust logging and error handling mechanisms

Technologies Used

  • C# and .NET for backend logic and services
  • MetaTrader 5 API for market integration
  • Asynchronous programming with async/await
  • Dependency Injection for scalable service management

Use Case

This strategy is perfect for developers and traders looking to automate a disciplined buy-side approach. It provides a scalable base for integrating more advanced features or building into a full trading system.
